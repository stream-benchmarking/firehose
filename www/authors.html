FireHose Authors

The authors of the FireHose suite are Karl Anderson (keander at sandia.gov) and Steve Plimpton (sjplimp at sandia.gov).

You can send us email regarding any of these issues:

  • If you use FireHose for something interesting, we'd like to hear about it.
  • If you encounter a bug or have a question about using the software, send us an email.
  • If you write a paper using the FireHose suite, we can post a citation to it on this page.
  • If you write a new analytic or generator that others would be interested in, we can include it in the distribution hosted on GitHub.
  • If you have new benchmarking results that others would be interested in, we can include it in the distribution hosted on GitHub.
